# Multithreading & Concurrency Tasks in C#

This project contains **10 practical examples** of multithreading, synchronization, and asynchronous programming in C#. It runs via an interactive console menu.

---

## 🚀 Features & Tasks Included

| # | Task | Core Concept Covered |
|---|---|---|
| 1 | **Parallel Counter** | Multi-threaded counting & race conditions. |
| 2 | **Sum Numbers** | Dividing heavy calculations across threads. |
| 3 | **Shared Bank Account** | Critical section protection using `lock`. |
| 4 | **Producer-Consumer** | Thread synchronization with shared buffers. |
| 5 | **Thread-Safe Logger** | Concurrent resource writing. |
| 6 | **Sequential Execution** | Execution ordering using `Thread.Join()`. |
| 7 | **Prime Number Finder** | Dynamic load distribution among threads. |
| 8 | **Background Autosave** | Background workers vs User input threads. |
| 9 | **Download Manager** | Simulating multiple concurrent downloads. |
| 10| **Ticket Booking** | Resource allocation & avoiding double-booking. |

---
